# FMCG-Retail-Data-Consolidation

•	Designed and implemented a scalable Lakehouse ETL pipeline using Databricks (Python, SQL, Spark) to unify data from a large retail enterprise and an acquired subsidiary into a Medallion Architecture on Amazon S3.

•	Automated ingestion, cleansing, and transformation workflows, reducing data reconciliation time by 40% and enabling unified analytics via BI dashboards and Genie for executive decision-making.
